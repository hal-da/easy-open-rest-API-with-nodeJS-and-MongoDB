Rest API
Test Customer Database for Webframeworks

  • GET /api - all customers
  • GET /api/id - single customer by id
  • POST /api - new customer
  • PATCH /api/id - change customers - send only the parts you like to change as json
  • DELETE /api/id - deletes single customer by id
